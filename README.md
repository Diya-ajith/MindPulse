# 🧠 MindPulse – Emotion Detection and Emotional Pressure Analysis

## 📖 Overview

**MindPulse** is an AI-powered **Emotion Detection and Emotional Pressure Analysis System** developed using **Natural Language Processing (NLP)** and **Deep Learning**. It utilizes a **fine-tuned BERT model** trained on the **GoEmotions dataset** to detect **28 distinct human emotions** from textual input.

The application predicts the dominant emotion, provides a confidence score, analyzes emotional pressure, generates context-aware responses, and visualizes emotion probabilities using interactive charts through a **Streamlit** interface.

---

## ✨ Features

- 🤖 Detects **28 human emotions** from text
- 🧠 Fine-tuned **BERT** model for emotion classification
- 📊 Confidence score prediction
- ❤️ Emotional pressure analysis
- 💬 Context-aware response generation
- 📈 Interactive pie chart and bar graph visualization
- 🌐 User-friendly Streamlit web interface
