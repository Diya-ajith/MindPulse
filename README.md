🧠 MindPulse – Emotion Detection and Emotional Pressure Analysis
📖 Overview
MindPulse is an AI-powered emotion detection system that uses a fine-tuned BERT model trained on the GoEmotions dataset to identify 28 human emotions from text. The application analyzes user input, predicts the dominant emotion, provides a confidence score, estimates emotional pressure, generates context-aware responses, and visualizes emotion probabilities through interactive charts.

✨ Features
🤖 Detects 28 emotions from text

🧠 Fine-tuned BERT model

📊 Confidence score prediction

❤️ Emotional pressure analysis

💬 Context-aware responses

📈 Pie chart & bar graph visualization

🌐 Interactive Streamlit interface
